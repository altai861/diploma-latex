\section{Системийн үйл ажиллагааны тухай дэлгэрэнгүй}

\paragraph{}
Хэрэглэгч үндсэн серверийн программыг өөрийн хүссэн серверийн компьютер дээр ажиллуулах бөгөөд сүлжээний хандалтын асуудлыг өөрөө шийдвэрлэнэ. Серверийн програмыг ажиллуулсны дараа хэрэглэгч системийн admin interface-д нэвтэрч анхны администратор (admin) хэрэглэгчийг үүсгэнэ. Мөн энэ үед системийн үндсэн үүлэн хадгалалтын хавтсыг (storage directory) зааж өгнө.

\paragraph{}
Анхны тохиргоо хийгдсэний дараа хэрэглэгч веб аппликейшн эсвэл iOS гар утасны аппликейшн ашиглан системд нэвтрэх боломжтой болно. Администратор хэрэглэгч нь системд шинэ хэрэглэгч үүсгэх, хэрэглэгчийн мэдээллийг удирдах, мөн хэрэглэгчдэд тодорхой хавтас болон файл дээр read эсвэл edit эрх олгох боломжтой байна.

\paragraph{}
Администратор хэрэглэгч нь системийн хадгалалтын орчинд дурын хавтас болон файлыг үүсгэх, засах, устгах, мөн файлуудыг download болон upload хийх бүрэн эрхтэй байна.

\paragraph{}
Энгийн хэрэглэгч нь администратороос олгосон эрхийн хүрээнд системд хадгалагдсан файлууд болон хавтаснуудтай ажиллах боломжтой бөгөөд файлыг харах, татаж авах, эсвэл засварлах зэрэг үйлдлүүдийг зөвшөөрөгдсөн хэмжээнд гүйцэтгэнэ.
% \section{Системийг ашиглах хэрэглэгчид}

\section{Функциональ шаардлага}

\begin{enumerate}

\item Серверийг анх ажиллуулах үед анхны админ хэрэглэгчийг үүсгэх боломжтой байх.

\item Үүлэн хадгалалтад ашиглах үндсэн хадгалалтын хавтсыг (storage folder) сонгон тохируулах боломжтой байх.

\item Хэрэглэгч веб аппликейшн болон iOS гар утасны аппликейшн ашиглан системд нэвтрэх боломжтой байх.

\item Админ шинэ хэрэглэгч үүсгэх боломжтой байх.

\item Админ хэрэглэгчийн нууц үгийг өөрчлөх болон хэрэглэгчийг устгах боломжтой байх.

\item Админ хэрэглэгчдэд тодорхой файл болон хавтас дээр read эсвэл edit эрх олгох боломжтой байх.

\item Хэрэглэгч өөрийн edit эрхтэй файл болон хавтас дээр бусад хэрэглэгчдэд эрх олгох боломжтой байх.

\item Хэрэглэгч системд хадгалагдсан файл болон хавтасны бүтцийг харах боломжтой байх.

\item Хэрэглэгч сервер рүү файл байршуулах (upload) боломжтой байх.

\item Хэрэглэгч системд хадгалагдсан файлыг татаж авах (download) боломжтой байх.

\item Хэрэглэгч шинэ хавтас үүсгэх боломжтой байх.

\item Хэрэглэгч файл болон хавтсын нэрийг өөрчлөх боломжтой байх.

\item Хэрэглэгч файл болон хавтсыг устгах боломжтой байх.

\item Хэрэглэгч файл болон хавтсыг өөр хавтас руу шилжүүлэх боломжтой байх.

\item Хэрэглэгч дараах төрлийн файлуудыг preview хийж харах боломжтой байх. Зураг (jpg, png, gif, webp), Text (txt, json, csv, md), Documents (pdf), Audio (mp3), Video (mp4)

\end{enumerate}

\section{Функциональ бус шаардлага}

\begin{enumerate}

\item Серверийн программ нь Linux, Windows, MacOS үйлдлийн системүүд дээр ажиллах боломжтой байх.

\item Хэрэглэгчийн client аппликейшн нь web болон iOS платформ дээр ажиллах боломжтой байх.

\item Web аппликейшн нь Chrome, Firefox, Safari болон Edge браузеруудын сүүлийн хоёр хувилбар дээр зөв ажиллах боломжтой байх.

\item Хэрэглэгч болон серверийн хооронд дамжих бүх өгөгдөл HTTPS (TLS) протокол ашиглан шифрлэгдсэн байх.

\item Хэрэглэгчийн нууц үгийг сервер дээр plaintext хэлбэрээр хадгалахгүй бөгөөд аюулгүй hash алгоритм ашиглан хадгалдаг байх.

\item Систем нь нэг сервер дээр олон хэрэглэгчийн зэрэг хандалтыг дэмжих боломжтой байх.

\item Файл байршуулах болон татах үйлдлийн явцыг хэрэглэгчийн интерфэйс дээр харуулах боломжтой байх.

\item Систем нь хэрэглэгч зөвхөн өөрийн файл болон өөрт олгогдсон эрхтэй өгөгдөлд хандах боломжийг хангах байх.

\item Файл дамжуулах явцад өгөгдлийн бүрэн бүтэн байдлыг шалгах механизм (checksum эсвэл hash) ашигладаг байх.

\end{enumerate}

\section{Ажлын хүрээ}

\subsection{Scope (Хамрах хүрээ)}

Энэхүү төслийн хүрээнд дараах програм хангамжийн бүрэлдэхүүнүүдийг хөгжүүлнэ:

\begin{enumerate}
    \item \textbf{Хувийн үүлэн хадгалалтын үндсэн серверийн программ (Main Server Application): } Үндсэн серверийн программ нь хэрэглэгчийн компьютерт суурилж, өгөгдлийг хадгалах, төхөөрөмж бүртгэх, холболтыг удирдах, файл дамжуулалтыг гүйцэтгэх үүрэгтэй байна.
    \item \textbf{iOS үйлдлийн системд зориулсан мобайл клиент аппликейшн: } iOS мобайл аппликейшн нь хувийн үүлэн хадгалалтын серверийн үндсэн үйлдлүүдийг хийх боломжтой байна. Нэвтрэх, файл болон хавтас үзэх, файл татах, файл байршуулах гэх мэт дээр оруулсан үйлдлүүдийг хийнэ.
    \item \textbf{Веб апп:} iOS апп-тай адилхан бүх үйлдлийг хийх боломжтой байна. Үндсэн үйлдлээс гадна. Хамгийн эхний тохигооны алхам буюу админ хэрэглэгч үүсгэх болон үндсэн хавтас сонгох алхамууд нэмэгдэж хийгдэнэ.
\end{enumerate}

\subsection{Out of Scope (Хийхгүй зүйлс)}

\begin{enumerate}
    \item \textbf{Android мобайл аппликейшн: }

    Android үйлдлийн системд зориулсан мобайл аппликейшн 
    энэхүү төслийн хүрээнд хөгжүүлэхгүй. Төслийн хүрээнд 
    нэг платформ дээр төвлөрч, native хөгжүүлэлтийг гүнзгий түвшинд 
    хийх зорилгоор iOS платформыг сонгосон болно.

\end{enumerate}

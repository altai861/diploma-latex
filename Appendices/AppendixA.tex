\chapter{Серверийн API-ийн жишээ}
\label{AppendixA}

\section{REST endpoint-ууд}
\paragraph{}
Энэ хавсралтад хувийн үүлэн хадгалалтын системийн сервер талын үндсэн API endpoint-уудын жишээг үзүүлэв.

\begin{lstlisting}
POST   /api/auth/login
POST   /api/auth/logout

GET    /api/users/me
POST   /api/users               // admin: create user
DELETE /api/users/{userId}      // admin: delete user

GET    /api/files?path=/        // list files/folders
POST   /api/files/upload        // upload file
GET    /api/files/download/{id} // download file
PATCH  /api/files/{id}/rename   // rename file/folder
DELETE /api/files/{id}          // delete file/folder

POST   /api/shares              // grant permission
DELETE /api/shares/{shareId}    // revoke permission
\end{lstlisting}
